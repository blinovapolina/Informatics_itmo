## Задачи
1. Установить виртуальную машину и скачать дистрибутив Ubuntu
2. Создать файл `script.bash`
3. Модифировать файл `script.bash` так, чтобы при его запуске в терминале в виде

```bash
bash script.bash Vasya Pupkin
```
